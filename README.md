# air-playmat

ポケモンTCGのデジタル対戦用プレイマットアプリ。ブラウザ上でカードの配置・操作をシミュレーションできます。

## 技術スタック

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- dnd-kit（ドラッグ&ドロップ）
- Vitest（テスト）
- Vercel（デプロイ）

## セットアップ

```bash
npm install
npm run dev
```

## コマンド

```bash
npm run dev        # 開発サーバー
npm run build      # TypeScript型チェック + ビルド
npm run lint       # ESLint
npm run test       # テスト実行
npm run test -- --coverage  # カバレッジ付き
```

## 機能

- **デッキインポート**: ポケモンカード公式サイトのデッキコードからデッキを読み込み
- **デッキ管理**: インポート済みデッキの一覧表示・詳細確認・選択・削除
- **対戦シミュレーション**: カードのドラッグ&ドロップによる配置・移動
- **カード操作**: 進化・エネルギー付与・ダメージカウンター・状態異常の管理
- **山札・トラッシュ操作**: カード一覧からの手札追加・山札戻しなど
- **コイントス**: 表/裏のランダム判定
- **ゲームログ**: 操作履歴の記録・表示
- **キーボードショートカット**: ドロー・シャッフル・コイントスなどの操作
