# 山札（デッキ）機能の実装完了・確認事項

## 実装した機能
計画通り、以下の機能を実装しました。

1. **山札のシャッフル機能**
   - `useGameState` に `shuffleDeck` を追加し、Boardのデッキエリア右上のShuffleアイコンから実行可能にしました。
2. **手札を山札の下に戻す／戻してシャッフルする機能**
   - 手札にあるカードをクリックした際のメニュー (`CardMenu.tsx`) に「山札の下に戻す」「山札に戻してシャッフル」ボタンを追加しました。
3. **山札のダメージなどのUI非表示化**
   - `Card.tsx` にて、`area === 'deck'` の場合はダメージカウンターや状態異常アイコンを描画せず、そもそもカードクリックによるメニュー（`CardMenu`）を開かないように制御しました。
4. **山札クリックによる1枚ドロー機能**
   - 山札のカードの束（`Deck`エリア）をクリックすると `drawCard('p1')` が呼ばれ、一番上のカードが手札に移動・表側状態になります。
5. **山札の中身を表側で一覧で見る機能**
   - デッキエリア右上のSearchアイコンをクリックすると、モーダル（`showDeckModal`）が立ち上がり、山札にあるすべてのカードを表向きで一覧表示します。

## ユーザー様への確認事項
自動ブラウザテスト環境が一時的にLinux環境以外（macOS）をサポートしていないため、エージェントによる自動UI操作テストが完遂できませんでした（コードのコンパイルとLintエラーチェックは通過しています）。

お手数をおかけしますが、お手元のブラウザ（`http://localhost:5173`）にて、以下の操作を実際にお試しいただき、期待通りの動作となっているか手動でご確認をお願いいたします。

- [ ] 手札のカードをクリックし、「山札の下に戻す」または「山札に戻してシャッフル」を押して、カードが山札に移動すること
- [ ] 山札の束をクリックして、1枚手札にドローできること
- [ ] 山札エリア右上の「虫眼鏡アイコン」をクリックして山札一覧のモーダルが開き、中身が見えること
- [ ] 山札エリア右上の「シャッフルアイコン」を押してエラーが出ないこと（内部状態のシャッフルのため見た目の変化は少ないかもしれません）
- [ ] 山札にあるカードには、ダメージや状態異常のUIが表示されていないこと
